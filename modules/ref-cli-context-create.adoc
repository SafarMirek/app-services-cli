= context create

[role="_abstract"]
Create a service context

[discrete]
== Synopsis

Create a service context and assign associated service identifiers

....
rhoas context create [flags]
....

[discrete]
== Examples

....
## Create context
$ rhoas context create --name dev

....

[discrete]
== Options

      `--name` _string_::   Name of the context
